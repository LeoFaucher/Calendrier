Projet Calendrier

Ce calendrier développé par des codeurs de la Coding Factory présente toute les fonctionnalités modernes recherchées: du Data Picker au récapitulatif d'évènement à venir, il arbore également un agenda pour y configurer vos journées dans le moindre détail.

Pour réaliser ce projet, notre cahier des charges était le suivant:

  • Un curseur pour la modification de mois
  • Une page listant les évements disponible
  • La possibilité de créer un évènement sur le calendrier
  • Un Data Picker pour la saisie de date
  • Le Responsive Design desktop/mobile/tablette
  • Un Burger Menu
  • Des psoeudo-elements choisi sur Font Awesome
  • Un thème rappelant ITESCIA
